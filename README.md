# Cómo votan Diputados y Senadores

Dos sitios para mirar, en claro, cómo vota el Congreso argentino:

- [diputados.argentinadatos.com](https://diputados.argentinadatos.com) — Cámara de Diputados
- [senadores.argentinadatos.com](https://senadores.argentinadatos.com) — Senado

## Qué podés hacer

- Ver cada votación: a favor, en contra, abstenciones y ausencias
- Entrar a la ficha de un diputado o senador y ver su historial
- Comparar bloques (Diputados) o partidos (Senado)
- Ver quién coincide más al votar y cuánto asiste a las sesiones

## De dónde salen los datos

Los datos vienen de las fuentes oficiales del Congreso y se publican de forma abierta a través de [argentinadatos.com](https://argentinadatos.com).

Sitios oficiales:

- [diputados.gob.ar](https://www.diputados.gob.ar/)
- [senadores.gob.ar](https://www.senadores.gob.ar/)
